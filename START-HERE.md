# 🎯 COMMENCER ICI

## Bienvenue! 👋

Vous avez maintenant un **système automatisé complet** pour créer et déployer des articles sans copier-coller!

---

## ⚡ Quickstart (5 minutes)

### 1. **Installez les dépendances** (une seule fois)

#### Windows
**Double-cliquez:** `start.bat` ← Fait tout automatiquement!

#### Mac/Linux
```bash
chmod +x start.sh
./start.sh
```

#### VS Code
Appuyez: **Ctrl+Shift+B** → "🚀 Démarrer Admin Server"

---

### 2. **Ouvrez le navigateur**
```
http://localhost:3000
```

---

### 3. **Créez un article**
1. Cliquez le bouton **`+`** rouge (bas à droite)
2. Remplissez le formulaire
3. Cliquez **"Créer & Déployer"**

**Voilà!** ✅ Tout est automatisé:
- Article créé dans `articles/`
- Image uploadée dans `images/`
- TOTAL_ARTICLES mis à jour
- Commité et pushé vers GitHub

## 🐛 Aide Rapide

|----------|----------|
| Rien ne s'affiche | Vérifiez http://localhost:3000 |
| Bouton `+` invisible | Scrollez bas à droite |
| Formulaire fermé | Cliquez `+` à nouveau |
| Erreur d'upload | Vérifiez le format image |
| Git error | Configurez Git d'abord |
| Port déjà utilisé | `PORT=5000 npm start` |
